# Workout library

153 structured rowing workouts generated by `tools/workouts/generate_workouts.py`,
designed with the project's own models: W'-balance simulation for interval floors
(Skiba 2012), cube-law power targets for rate ladders, Beta "technique"/"race"
preset bands for form and race sessions.

- `xsr_format/` — EXR raw JSON, importable in the EXR web profile
- `zwo_format/` — Zwift workout XML, importable in Zwift and compatible platforms

Categories: Technique & Form, Endurance, Threshold, W-Prime Intervals,
Rate Ladders, Race Prep, HIIT, Recovery. Reference athlete for the W'
predictions in the descriptions: CP = FTP = 200 W, W' = 12 kJ.
